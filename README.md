# E-Commerce Front-End

A fully responsive, highly polished, and premium e-commerce front-end application built using Vanilla HTML, CSS, and JavaScript. This project showcases a modern user interface designed to provide an optimal shopping experience.

---

## 🚀 Live Demo & Preview
This project consists of 4 main page layouts designed with custom typography, clean spacing, and modern UI elements.

- **Homepage (`homepage.html`)**: Features product categories, trending hero banner, limited-time deals, newsletter subscription, and region-wise suppliers.
- **Product List (`products.html`)**: Includes an advanced sidebar filter, grid/list layout toggle, sorting, pagination, and a wishlist system.
- **Product Details (`productdetails.html`)**: Displays detailed information for individual products, gallery previews, specifications, and checkout options.
- **Shopping Cart (`cart.html`)**: An overview of selected items, pricing breakdowns, and checkout actions.

---

## ✨ Key Features
- **Modern Grid/List Layout Toggle**: Switch dynamically between grid and list views on the product listing page.
- **Interactive Wishlist System**: Toggle items in and out of your wishlist with micro-animations.
- **Advanced Sidebar Filtering**: Filter by category, brand, features, condition, price range, and star ratings.
- **Fully Responsive Design**: Optimized for mobile, tablet, and desktop viewports.
- **Clean Animations**: Soft hover states, sliding category menus, and input glows.

---

## 🛠️ Technologies Used
- **Structure**: Semantic HTML5
- **Styling**: Vanilla CSS3 (Custom grids, flexbox layout, and CSS variables)
- **Interactions**: Vanilla JavaScript (DOM manipulation, layout state toggles, and UI event handlers)
- **Icons**: SVG vectors for pixel-perfect clarity

## 💻 How to Run Locally

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/alihassansumbal/E-commerce-front-end.git
   ```
2. **Navigate to the Project Folder:**
   ```bash
   cd E-commerce-front-end
   ```
3. **Open the App:**
   Simply open `homepage.html` in any web browser, or use a local dev server like VS Code's **Live Server** for the best experience.
